BattleGrid

role: architect
role: engineer
role: artist
role: animator
language: java
BattleGrid was a reproduction of the combat system in Capcom's "Mega Man Battle Network" video game series which ran from March 2001 until it's cancellation in November of 2009. BattleGrid has not been maintained in many years, but the code and design notes where shared with projects like MMDBN: ChronoX and Tri Wing which are both fantastic spiritual successors to the original game series.
